# 线性代数学习要点

线性代数研究向量、矩阵和线性变换之间的关系。理解这些对象的结构，比只记住计算步骤更有助于迁移到机器学习和数据分析问题。

## 线性无关

一组向量线性无关，表示其中任何一个向量都不能由其余向量的线性组合表示。判断线性无关时，可以把向量组成矩阵，并分析齐次方程是否只有零解。

## 基与维数

向量空间的一组基既要线性无关，也要能够张成整个空间。任意一组基所含向量的数量相同，这个数量称为向量空间的维数。

# Anki 主动回忆流程

主动回忆要求学习者在看到答案前先尝试提取记忆。卡片问题应聚焦单一知识点，答案应直接回应问题，并保留必要的限制条件或易混点。

## 人工审核

AI 生成的候选卡片必须经过人工审核。审核者需要检查事实准确性、问题是否原子化，以及答案是否适合长期复习；未经明确批准的候选卡片不能进入后续写入阶段。
