## Manual `.ankiaddon` install

For an explicitly provided candidate package, open **Tools → Add-ons**, use **Install from file**, select `ankiforge_ai.ankiaddon`, and restart Anki. Verify the package source, SHA-256, version, and manual-acceptance status first. Never install a package containing config, logs, backups, credentials, or Anki collection files.

## Common installation problems

- **Cannot find it in Shared Decks:** use Get Add-ons and the code above.
- **Old interface after update:** restart Anki and confirm the add-on version.
- **Add-on fails to load:** record Anki/add-on versions and the safe error message; do not publish API keys or personal collection data.
