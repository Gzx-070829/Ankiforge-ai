# 通过 AnkiWeb 安装 AnkiForge AI

AnkiWeb 插件代码：`1227582295`

AnkiWeb 页面：[https://ankiweb.net/shared/info/1227582295](https://ankiweb.net/shared/info/1227582295)

## 安装步骤

1. 打开 Anki。
2. 选择“工具 → 插件 → 获取插件”。
3. 输入插件代码 `1227582295`。
4. 确认安装，完成后重启 Anki。
5. 打开 AnkiForge AI。

如需从本地安装包安装，请参阅[从 `.ankiaddon` 文件手动安装](manual_install_from_ankiaddon.md)。
