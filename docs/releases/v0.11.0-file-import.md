# AnkiForge AI v0.11.0 文件导入与界面优化

状态：Release candidate — AnkiWeb 更新材料已准备

AnkiWeb add-on code：`1227582295`

## 更新内容

- 学习材料区域支持拖入本地文件
- 文件选择按钮支持 Markdown、TXT、DOCX 和 PDF 入口
- Markdown / TXT 保留原始文本结构
- DOCX 使用纯 Python 提取段落和简单表格文本
- 导入成功后显示文件名、类型和字符数
- 多文件拖入只处理第一个并显示提示
- 已有材料会安全追加，不会静默覆盖
- 导入区域、Provider / Model / API key 区域和 Anki 写入配置层级更清晰
- 导入成功、警告与错误提示更易区分
- 中英文界面同步更新

## PDF 状态

本构建的 PDF 为 fallback-only：未捆绑 PDF 解析器，也不包含 OCR。选择或拖入 PDF 时，插件会友好提示复制可选文本或先转换为 TXT / Markdown。不会宣称支持扫描版 PDF 或完整保留复杂论文排版。

## 保持不变的安全边界

- 文件导入不会自动调用 AI
- API key 只在当前会话使用，不保存
- 不会自动写入 Anki
- 写入前仍必须审核、查重并二次确认
- 不修改已有卡片、牌组或笔记类型

可直接粘贴到 AnkiWeb 的完整中英文描述见 [AnkiWeb v0.11 描述草稿](../ankiweb_description_v0_11.md)。
