# AnkiForge AI v0.11 `.ankiaddon` 手工验收

请在独立 Anki 测试配置和测试牌组中执行；不要使用重要个人 collection。

## 安装与界面

- [ ] 从文件安装最终 `dist/ankiforge_ai.ankiaddon`，重启后插件正常加载
- [ ] 主界面保持两栏布局，窗口缩放后控件无明显重叠
- [ ] 学习材料拖放区域清晰，中文与英文切换正常
- [ ] Provider、Model、API key 和右侧写入配置区信息层级清楚

## 文件导入

- [ ] 选择和拖放 `.md`、`.txt`、`.docx` 均可导入
- [ ] 成功提示包含文件名、类型和字符数
- [ ] 多文件拖放只处理第一个，并显示说明
- [ ] 原有文本不会被静默覆盖，新材料追加到末尾
- [ ] DOCX 只提取段落和简单表格文本
- [ ] PDF 显示 fallback 指引，不执行解析、OCR 或联网

## 安全边界

- [ ] 导入文件不会调用 Provider
- [ ] API key 只在当前窗口使用，关闭后不保存
- [ ] 未点击生成时不产生网络请求
- [ ] AI 生成结果必须审核，写入前必须最终确认
- [ ] 取消确认不会写入卡片
- [ ] 不修改已有卡片、牌组或笔记类型

## 发布包

- [ ] 包内没有 config、backup、secret、缓存、测试、文档或 Anki 用户数据
- [ ] 文件数量、大小和 SHA-256 与发布记录一致
- [ ] 两次从相同 main 源码构建得到相同 SHA-256
